# 📘 Interactive Vocabulary

An interactive English vocabulary learning web app with pronunciation, search, and save-word features.

## ✨ Features

- Dynamic lesson buttons from API
- Vocabulary cards by lesson
- Word pronunciation using Speech Synthesis API
- Search functionality
- Save favorite words
- Responsive design
- Loading spinner
- Modal with detailed vocabulary info

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Tailwind CSS
- DaisyUI
- REST API
