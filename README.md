# Interactive Vocabulary

Interactive Vocabulary is a responsive vocabulary learning web application that allows users to explore lessons, search words instantly, and view detailed vocabulary information through an interactive interface.

## Live Demo
https://mohammedrakibcodes.github.io/Interactive-vocabulary/

## Features
- Dynamic lesson loading from API
- Real-time vocabulary search
- Detailed vocabulary modal
- Loading spinner and empty-state handling
- Responsive card-based layout
- Responsive design for all devices

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- DaisyUI
- REST API

## Run Locally

```bash
git clone https://github.com/mohammedrakibcodes/Interactive-vocabulary.git

cd Interactive-vocabulary
```
