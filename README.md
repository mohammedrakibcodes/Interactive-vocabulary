# 📘 Interactive Vocabulary

Interactive Vocabulary is a modern English vocabulary learning web application where users can explore vocabulary lessons, search words instantly, view detailed meanings, and improve pronunciation through an interactive learning experience.

---

## ✨ Features

- Dynamic lesson buttons generated from API
- Vocabulary loading based on selected lesson
- Real-time search functionality
- Responsive card-based vocabulary layout
- Detailed vocabulary modal
- Loading spinner while fetching API data
- Active lesson button state
- Empty-state handling for unavailable lessons
- Responsive design for mobile, tablet, and desktop
- Modern UI built with Tailwind CSS & DaisyUI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- DaisyUI
- Font Awesome
- REST API

---

## 🔍 Core Functionalities

### Vocabulary Lessons
Users can load vocabulary words dynamically by selecting lesson buttons.

### Search System
Users can search vocabulary words instantly using the search bar.

### Vocabulary Details Modal
Clicking the info button opens a modal containing:

- Word
- Pronunciation
- Meaning
- Example Sentence
- Synonyms

### Loading State
Skeleton loading spinner appears while vocabulary data is loading.

### Error & Empty State Handling
Displays user-friendly messages when vocabulary data is unavailable.
